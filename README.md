Per motivi di dimensioni, i dataset originali, i modelli YOLO e tutti i risultati completi non sono stati caricati su GitHub.

Questo include:
- Modelli YOLO (*.pt)
- Dataset finale per il modello, con opportune sezioni (dataset_combined/)
- Dati necessari. Includono le sequenze video iniziali, i frames derivanti, con una sezione per quelli annotati, e le detections per ogni frame, con una sezione per quelle relative alla parte di test (data/)
- Output intermedi e finali dei vari approcci, fatta eccezione per il modello fine-tuned (debug/)
- Output finali del modello fine-tuned (runs/)


Risultati esemplificativi sono disponibili al seguente link Google Drive:
[Link to the Drive folder](https://drive.google.com/drive/folders/1dTUx-vc6fisWk62uoVUkqJzUdJ_kvCzc?usp=drive_link)

- Output finali dei vari approcci, fatta eccezione per il modello fine-tuned, eslcusivamente per il mio primo frame relativo al test su cui sono state calcoalte le metriche (debug_sample/)
- Output finali del modello fine-tuned, sul medesimo frame per le sequenze trattate e su un sottoinsieme di frames per il nuovo dataset della Bundesliga (runs_sample/)
